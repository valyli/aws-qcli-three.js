# Three.js 赛车小游戏开发计划

## 项目概述
创建一个简单的 3D 赛车游戏，玩家可以控制一辆车在赛道上行驶。游戏将使用 Three.js 进行 3D 渲染，并实现基本的物理效果和碰撞检测。

## 开发环境
- HTML5 + CSS3 + JavaScript
- Three.js 库
- 简单的项目结构，无需复杂构建工具
